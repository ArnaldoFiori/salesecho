# Etapa 07 — Frontend: Portal Completo

## Objetivo
Todas as telas do portal do gestor consumindo a API backend.

## Spec de referência
- `spec/04_PORTAL_GESTOR.md`

## Entregável
- Dashboard: 4 cards stats, últimos 10 registros, banners trial/inadimplência
- Recordings: listagem com filtros (seller, customer, período, status), paginação 20/page
- Recording Detail: transcrição + resumo completo
- Export XLSX (botão download)
- Sellers: listagem, adicionar, editar, toggle ativo/inativo, desvincular Telegram
- Account: dados empresa/gestor, botão "Gerenciar Assinatura" (Stripe Portal), alterar senha
- Admin (system_admin): lista orgs, métricas operacionais

## Validação
Navegar todas as telas com dados reais. Export XLSX funcional.
